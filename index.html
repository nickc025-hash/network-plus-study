Subnetting Tips

Quick Math: Number of hosts = 2^(32-CIDR) - 2
Common Masks:
/24 = 255.255.255.0 (256 hosts)
/25 = 255.255.255.128 (128 hosts)
/26 = 255.255.255.192 (64 hosts)
/27 = 255.255.255.224 (32 hosts)
/28 = 255.255.255.240 (16 hosts)
Remember: Always subtract 2 from total hosts (network & broadcast addresses)
Block Sizes:
/24 = 256 block size
/25 = 128 block size
/26 = 64 block size
/27 = 32 block size
/28 = 16 block size

Access Control List Configuration

Scenario:

Configure an ACL for a router with the following requirements:

  • Allow HTTP traffic from 192.168.1.0/24 to any destination
  • Deny FTP traffic from 10.0.0.0/8
  • Allow SSH from 172.16.5.10 only
  • Deny all other traffic

Configure ACL Entries:

Correct ACL Configuration:

access-list 100 permit tcp 192.168.1.0 0.0.0.255 any eq 80
access-list 100 deny tcp 10.0.0.0 0.255.255.255 any eq 21
access-list 100 permit tcp host 172.16.5.10 any eq 22
access-list 100 deny ip any any
💡

ACL Configuration Tips

Wildcard Masks:
Invert the subnet mask bits
255.255.255.0 → 0.0.0.255
255.255.0.0 → 0.0.255.255
Common Ports:
HTTP = 80
HTTPS = 443
FTP = 21
SSH = 22
Telnet = 23
ACL Order: Place most specific rules first, general rules last
Keywords:
'host' = single IP (0.0.0.0 mask)
'any' = all IPs (255.255.255.255)
Remember: Standard ACLs (1-99) filter by source IP only. Extended ACLs (100-199) filter by source, destination, and port

Cable Facts & Tips

  • 💡 Remember: Cat6 can do 10Gbps but only up to 55 meters!
  • 💡 Tip: Single-mode fiber uses lasers, multi-mode uses LEDs
  • 💡 Quick fact: RJ stands for "Registered Jack"
  • 💡 Memory aid: LC = "Lucent Connector" or "Little Connector"
  • 💡 Important: Plenum-rated cables are required in air handling spaces
