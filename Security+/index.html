Wireshark Tips

TCP Flags:
SYN = Connection request
ACK = Acknowledgment
FIN = Connection termination
RST = Connection reset
Common Attack Patterns:
• Many SYN packets = SYN flood
• SQL keywords in HTTP = SQL injection
• Rapid port connections = Port scan
• Unusual DNS queries = C2 traffic
Protocol Hierarchy:
Layer 2: Ethernet
Layer 3: IP
Layer 4: TCP/UDP
Layer 7: HTTP/HTTPS/DNS
Red Flags:
• Cleartext passwords
• Malformed packets
• Unusual ports
• Base64 encoded data
• Non-standard protocols

Office Security Best Practices

Network Segmentation:
• Separate VLANs per department
• Firewall between segments
• Restrict inter-VLAN routing
Access Control:
• Least privilege principle
• Role-based permissions
• Regular access reviews
• MFA for sensitive areas

HIPAA: Healthcare data
GDPR: Personal data privacy
SOX: Financial reporting
Zero Trust Principles:
• Never trust, always verify
• Assume breach mentality
• Continuous validation
• Minimize attack surface
